# Fit-Smart
Proyek Fit-Smart berfokus pada pembuatan aplikasi yang dapat memberikan rekomendasi makanan dan latihan berdasarkan data pengguna seperti usia, tinggi badan, berat badan, dan tujuan (menurunkan atau menaikkan berat badan). Cakupan proyek meliputi pembuatan sistem input data, perhitungan BMI, serta sistem rekomendasi sederhana berbasis data.
