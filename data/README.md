# 📊 FitSmart - Nutrition Dataset

Dataset ini (`nutrition.csv`) memuat informasi nutrisi dasar dari berbagai jenis makanan dan minuman yang digunakan oleh sistem AI FitSmart untuk kalkulasi kalori harian pengguna.

## 📖 Data Dictionary

| Nama Kolom     | Tipe Data | Deskripsi |
|---------------|-----------|-----------|
| `name`        | String    | Nama jenis makanan atau minuman. |
| `calories`    | Float/Int | Jumlah energi atau kalori yang terkandung (dalam kkal). |
| `proteins`    | Float     | Kandungan makronutrien protein (dalam gram). |
| `fat`         | Float     | Kandungan makronutrien lemak total (dalam gram). |
| `carbohydrate`| Float     | Kandungan makronutrien karbohidrat (dalam gram). |
| `image`       | String    | Tautan URL atau *path* lokal menuju gambar referensi makanan tersebut (Opsional). |

---
*Dataset ini dibersihkan dan diproses sebagai bagian dari tahap Data Wrangling proyek FitSmart.*
